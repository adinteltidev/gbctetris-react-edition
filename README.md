# GBC Tetris React Edition

## Project Description
GBC Tetris is a classic Tetris game clone built with React. The game brings the nostalgic experience of playing Tetris on a Game Boy Color to modern web browsers, employing responsive design and fluid gameplay mechanics.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/adinteltidev/gbctetris-react-edition.git
   ```
2. Navigate to the project directory:
   ```
   cd gbctetris-react-edition
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Gameplay Instructions
- Use the arrow keys to move and rotate the Tetrimino pieces.
- The goal is to complete horizontal lines by filling them with Tetriminos.
- When a line is completed, it will disappear, and you earn points.
- The game ends when the stack of pieces reaches the top of the screen. 

Enjoy playing the game!
